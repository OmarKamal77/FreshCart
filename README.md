# 🛒 FreshCart Project

A fully responsive **E-commerce web application** built using **React.js, Vite, Tailwind CSS, and Context API**.  
FreshCart allows users to browse products, manage their cart and wishlist, and enjoy a smooth shopping experience.

---

## 🌟 Features
- ✅ Responsive design for all devices
- 🛍️ Product listing
- ❤️ Wishlist management (add / remove products)
- 🛒 Cart management with real-time updates
- 🔍 Product details page
- 🗂️ Categories & brands
- 🔑 Authentication pages (Login & Register UI)
- 🔔 Toast notifications using React Toastify

---

## 🧰 Technologies Used
- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Context API
- Font Awesome
- React Toastify
